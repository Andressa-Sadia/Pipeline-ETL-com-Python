# Pipeline ETL com Python

Neste projeto foi desenvolvido um pipeline ETL (Extract, Transform, Load) utilizando a linguagem Python, com o objetivo de compreender, na prática, o fluxo de dados entre as etapas de extração, transformação e carregamento, conforme proposto no desafio.

### Extração (Extract)
A etapa de extração foi realizada a partir de um arquivo CSV contendo dados fictícios de usuários, como nome, conta e cartão. Essa abordagem substitui o uso de uma API externa, permitindo a execução completa do projeto sem dependências externas.

## Transformação (Transform)
Na etapa de transformação, os dados extraídos foram processados por meio de regras definidas em Python, gerando mensagens personalizadas de marketing para cada usuário. Essa etapa simula a aplicação de Inteligência Artificial na geração de valor a partir dos dados.

## Carregamento (Load)
Por fim, os dados transformados foram carregados em um novo arquivo CSV, concluindo o ciclo do processo ETL e disponibilizando os dados prontos para uso.

## Considerações Finais
O projeto atende ao objetivo do Lab, demonstrando de forma clara o funcionamento do fluxo ETL, independentemente da ferramenta utilizada para extração dos dados.
